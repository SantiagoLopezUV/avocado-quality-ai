# MVP – Fruit Quality AI

## Objetivo del MVP
Permitir que un usuario suba una imagen de un aguacate y reciba una
clasificación básica de calidad.

## Incluye
- Subida de una imagen
- Procesamiento básico de la imagen
- Respuesta en formato JSON
- Clasificación simple de calidad

## No incluye
- Frontend avanzado
- Base de datos
- Autenticación
- Docker (se integrará en sprints posteriores)
